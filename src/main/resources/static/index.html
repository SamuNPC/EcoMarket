Endpoints Disponibles

Productos

/api/productos

Gestión completa de productos del catálogo

GET Obtener todos los productos
GET Obtener producto por ID
POST Crear nuevo producto
PUT Actualizar producto
DELETE Eliminar producto

Ejemplo POST:

{
  "nombreProducto": "Manzanas Orgánicas",
  "stock": 50
}
Ver en Swagger

Clientes

Administración de clientes registrados

GET Obtener todos los clientes

Guía de Inicio Rápido

1

Abre Swagger UI

Haz clic en el botón "Abrir Swagger UI" para acceder a la documentación interactiva.

2

Explora los Endpoints

Navega por los diferentes controladores para ver todos los endpoints disponibles.

3

Prueba las Operaciones

Usa el botón "Try it out" en Swagger para probar cada endpoint con datos reales.

4

Revisa las Respuestas

Examina las respuestas y códigos de estado para entender el comportamiento de la API.
